# lemonPPT 社区行为准则

## 我们的承诺

为了营造一个开放、友好、包容的社区环境，lemonPPT 项目的维护者和贡献者承诺：无论年龄、体型、残障、种族、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何，所有人都能在我们的社区中获得无骚扰的参与体验。

## 我们的标准

有助于营造积极环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点和经验
- 乐于接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表达同理心

不可接受的行为包括：

- 使用性暗示语言或图像，以及不受欢迎的性关注或挑逗
- 恶意挑衅、侮辱性评论、人身攻击或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息（如物理地址或电子邮箱）
- 其他在专业环境中可被合理视为不恰当的行为

## 维护者职责

项目维护者有责任澄清可接受行为的标准，并在出现不当行为时采取适当、公平的纠正措施。

维护者有权并有责任删除、编辑或拒绝不符合本准则的评论、提交、代码、Wiki 编辑、Issue 以及其他贡献，也可暂时或永久封禁任何他们认为不适当、威胁、冒犯或有害的贡献者。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共场所代表项目或社区时的行为。代表项目的例子包括使用官方电子邮箱、通过官方社交媒体账号发帖、或在在线或线下活动中担任指定代表。

## 执行

如果你遭遇了滥用、骚扰或其他不可接受的行为，请联系项目维护者。所有投诉将被及时、公平地审查和调查。

维护者有义务对举报事件保密。

## 参考
